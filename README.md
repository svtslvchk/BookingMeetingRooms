# Система бронирования переговорных комнат

Бэкенд-приложение на FastAPI для управления и бронирования переговорных комнат в режиме реального времени. В качестве СУБД используется PostgreSQL, взаимодействие с которой настроено через асинхронный инструмент SQLAlchemy 2.0 и систему миграций Alembic.
